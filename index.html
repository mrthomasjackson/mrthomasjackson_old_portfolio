SimplyEyeglasses.com Data Warehouse

About

This project allowed the company decision makers to gain insight into the company's transactions. This insight allows to make clearer decisions through reports and data visualization. This system also allowed the business to isolate pain-points in pricing, and transactions.

The system was developed using MVC architecture through the PHP Laravel 5 framework. Raw data is imported from their point of sale system using XML files. The XML files are parsed and entered into a MySQL database using PHP scripts.

MySQL queries are run to create HTML5 reports.

Technologies Used

PHP Laravel 5 MySQL Unix Vagrant HTML CSS JavaScript jQuery
